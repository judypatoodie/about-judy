About.

Hi there! My name is Judy and I am an avid learner and problem solver at heart. I enjoy being challenged, working collaboratively, and having opportunities to explore my creative avenues. Efficiency is one of my life mottos- that is why I love finding and learning new technologies because in one way or another, every piece of technology promotes efficiency in our everyday lives.

After graduating San Francisco State University with a Bachelor's degree in Information Systems, I worked as a Product Support Analyst before transitioning into a Developer/DevOps Support Engineer. Having gained exposure to Software Engineering when working to solve software bugs, I quickly realized my profound interest in all things Software. After, I moved on to being an Application Developer, developing custom solutions for internal stakeholders. To further my development skills, I decided to get my Web Development Certification with Bloc's immersive Full Stack Web Dev apprenticeship.

Projects.

Just Jams
This is a web based music player built using React.js, HTML, CSS, and Bootstrap.

Chitter Box
This is a web based, real-time chat application built using React.js. It is using Firebase as it's database, connecting via it's API. Users can sign-in using Google Authentication with Google's API.

One Market Network Support Portal
This is the company's Support Portal that I developed using JavaScript, HTML, and CSS. In this portal you can find FAQ's, Knowledge base articles, Runbooks, and forms you can use to submit a bug, question, or feature request. This is hosted on Zendesk.
Brain Share -
Brainshare is a article (wiki) posting and sharing application with a PostGreSQL database that stores data for Users, Collaborators, Wikis, etc. Markdown is enabled for new article posts. There is also a sign-up feature users can use to create an account. Stripe is integrated via their API for users to upgrade their accounts to premium to get access to more features in the application.

Brows By Chia website (work in progress)
This is a Website for a Powder Ombre Eyebrow Service. I designed the logo and built the wireframe using Adobe Photoshop. It has a PostGreSQL database to store section details and user information. It has a Calendly API for users to book appointments. This is built using Node.js, Express, HTML, and CSS.
